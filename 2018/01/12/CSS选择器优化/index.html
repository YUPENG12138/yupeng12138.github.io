如果存在以下情况，建议单独起类名，

2.使用 class 代替层级关系

3.利用缩写、继承、合并。

4.避免不必要的限制

注：在大型网站中，上面这种写法仍然不好，依据CSS解析从右向左的逻辑，会先解析页面中全部的a，再去查找#nav，所以可以考虑合理的使用class。

5.避免使用通用选择器
