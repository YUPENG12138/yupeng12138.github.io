webpack.config.js文件配置

  • entry 是一个对象，程序的入口
    • key:随意写
    • value: 入口文件
  • output 是一个对象,产出的资源
    • key: filename
    • value : 生成的build.js
  • module 模块（对象）
    • loaders:[]
      • 存在一些loader { test:正则,loader:'style-loader!css-loader' }
